# Gradius - Frontend

Frontend repository for the Gradius Degree Manager application. This interactive web application is built with **Next.js**, **Tailwind CSS**, and **React Flow** to visualize and manage degree paths.

## Features

- **Interactive Map**: Visualize subjects and their prerequisites as an interactive node graph using React Flow.
- **Status Toggling**: Click on any subject card to directly toggle its status between Pending, Regular, and Approved.
- **Dynamic Summaries**: Real-time dashboard headers reflecting overall degree progress.
- **Path Uploads**: User-friendly drag-and-drop mechanics to initialize degree datasets.

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Development

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
